Интернет магазин.
Данный интернет магазин разработан специально для продажи меда и прочего из этой тематики.
Состоит из 7 страниц:
* Главная - на которой отображаются 2 меню для удобства пользователя перемещения по сайту,
	    слайдер из 3 слайдов на каждом из которых отображается соостветствующая тематики
	    слайда ссылка на страницу сайта, последние добавленные товары и правый Sidebar для удобства пользователя.
* О себе  - для отображения информации о владельце магазина.
* Доставка - для отображения информации об условиях приобретения и доставки товаров(условия соглашения).
* Личный кабинет - для отслеживания своих заказов, адресов(платежного/доставки).
* Оплата - осуществляется 4-я видами: прямой банковский перевод, чековые платежи, оплата при доставке и Paymaster[UA].
* Магазин
* Корзина
Осуществляется благодаря плагину WooCommerce и другим.
